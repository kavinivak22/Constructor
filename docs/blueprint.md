# **App Name**: Constructor PWA

## Core Features:

- Multi-Project Dashboard: Provides a visual overview of all projects, key metrics, and quick actions.
- Material Management: Manage inventory levels and track consumption history for each project.
- Purchase Order Automation: Create, submit, and approve purchase orders with automated notifications.
- AI-Assisted Material Estimation: Use OpenAI to generate material estimations based on project specifications and historical data.
- Daily Worklog: Log attendance, hours, tasks, and material usage for each worker daily, including photo attachments. Material consumption automatically updates project inventory.
- Team Communication Hub: Facilitate real-time chat, file sharing, and notifications for all project teams.
- Offline Capability: Enable offline field usage via Service Worker and IndexedDB, syncing data when online.
- Personal Pouch: Users can store personal documents and notes.
- Project Pouch: Shared document and note storage for users assigned to a project.
- Employee Management: Manage employee accounts (add, edit, inactivate), assign/remove project access, and manage roles.

## Style Guidelines:

- Primary color: Vivid coral (#FF6B35) to reflect construction and action.
- Background color: Light grey (#E5E7EB) for a clean, professional look.
- Accent color: Fresh green (#27AE60) to indicate approvals or positive actions.
- Headings: 'Inter' sans-serif for clear headings. Note: currently only Google Fonts are supported.
- Body: 'Source Sans Pro' sans-serif for readable text. Note: currently only Google Fonts are supported.
- Use clear, consistent icons representing different actions, such as material tracking and expense management.
- Design large buttons and high-contrast elements for easy use with gloves, focusing on mobile-first responsiveness.
